# UX Assessment Plan: MikroTik Academy V2

## Scope
- **Pages/Flows:** MikroTik Academy Landing Page (`src/app/program/mikrotik/page.tsx`)
- **User Personas:** 
  - Calon siswa SMK yang ingin belajar jaringan komputer
  - Orang tua yang mencari program pendidikan teknologi berkualitas
  - Young professionals yang ingin sertifikasi MTCNA

## Assessment Criteria
- [ ] Visual Design Consistency (brand colors, typography, spacing)
- [ ] Responsive Behavior (mobile, tablet, desktop breakpoints)
- [ ] Usability Heuristics (Nielsen's 10 Principles)
- [ ] Placeholder Image Compliance (bg-gray-200 + text)
- [ ] Grid Module Responsiveness
- [ ] WCAG 2.1 AA Accessibility

## Methodology
1. Heuristic evaluation against Nielsen's 10 usability principles
2. Responsive breakpoint testing (320px, 768px, 1024px, 1280px)
3. Visual consistency check against brand tokens
4. Accessibility check (color contrast, focus visibility, semantic HTML)
5. Touch target size verification for mobile
